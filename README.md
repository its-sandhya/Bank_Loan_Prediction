# Bank Loan Prediction Web App

Predict whether a bank loan will be approved based on applicant details using **Random Forest** and **Flask**.

##  Project Overview
This project demonstrates a complete workflow:  
1. Data cleaning and preprocessing  
2. Feature importance analysis using Random Forest  
3. Model building with top 4 features  
4. Flask web application development  
5. Deployment on **Render**  

##  Tech Stack
- **Backend:** Python, Flask  
- **Modeling:** Scikit-learn, Pandas, Numpy  
- **Visualization:** Matplotlib, Seaborn  
- **Deployment:** Render
